Write a JavaScript program to check a set of three numbers; if the three numbers are the same return 30; otherwise return 20; and if two numbers are the same return 40. Viết chương trình JavaScript để kiểm tra bộ ba số; nếu ba số giống nhau thì trả về 30; nếu không thì trả về 20; và nếu hai số giống nhau thì trả về 40.
