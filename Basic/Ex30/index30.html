Write a JavaScript program to check whether a string "Script" appears at the 5th (index 4) position in a given string. If "Script" appears in the string, return the string without "Script" otherwise return the original one.

Viết chương trình JavaScript để kiểm tra xem chuỗi "Script" có xuất hiện ở vị trí thứ 5 (chỉ mục 4) trong một chuỗi nhất định hay không. Nếu "Script" xuất hiện trong chuỗi, hãy trả về chuỗi không có "Script", nếu không thì trả về chuỗi gốc.
